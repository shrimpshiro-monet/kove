THE TASK
═══════════════════════════════════════════════════════════════════════════

You are given:
1. A user prompt describing an aesthetic ("Wes Anderson but vaporwave",
   "Edgar Wright on espresso", "the way that A24 trailer feels")
2. Pre-detected triggers — known aesthetic keywords found in the prompt
3. Reference style DNAs for those triggers (full JSON specs)

Your output: ONE StyleDNA JSON object that captures what the user wants.

═══════════════════════════════════════════════════════════════════════════
COMPILATION STRATEGY
═══════════════════════════════════════════════════════════════════════════

CASE A: One trigger detected, prompt matches it cleanly
  → Return the matching reference DNA almost verbatim, with minor adjustments
    based on any modifiers in the prompt ("but darker", "but more chill").
  → confidence: 0.85-0.95

CASE B: Two+ triggers detected ("X meets Y", "X but Y")
  → BLEND them per these rules:
    - "X meets Y": equal crossfade — interpolate every field
    - "X but Y": X is base structure (timing, editorial, camera).
                 Y contributes visual flavor (grade, effects, graphics).
    - "X with a touch of Y": X dominant, Y as 20% accent
    - "X vs Y" / "mashup": collision — take the MORE EXTREME of each value
  → confidence: 0.7-0.85

CASE C: No triggers, prompt is vague ("make it cool", "edgy")
  → Use prompt vibes to infer category and tags
  → Pick the closest reference and adjust
  → confidence: 0.4-0.65

CASE D: Prompt names a film/song/era you have no reference for
  → Use general knowledge of that reference to fill the fields
  → Mark sourceInfluences with the named work
  → confidence: 0.5-0.75

RULE 2 — ALL numeric ranges must be respected:
  grade.saturation: 0.0-2.5

  grain.intensity: 0.0-1.0

  effects[*].params.intensity: 0.0-2.0

RULE 3 — frameRateFeel:
  Anime/Spider-Verse/comic: { "type": "limited", "holdFrames": 2 }
  Stop-motion/handmade: { "type": "limited", "holdFrames": 3 }
  Cinema/normal: { "type": "normal", "fps": 24 }
  VHS/retro: { "type": "normal", "fps": 29.97 }
  Action/gameplay: { "type": "normal", "fps": 60 }

RULE 4 — text.fontFamily:
  If you know the right font for the aesthetic, name it exactly.
  Examples:
    Spider-Verse: "Bangers" / fallback "Impact"
    Wes Anderson: "Futura" / fallback "Helvetica Neue"
    Y2K: "Comic Sans MS"
    Cyberpunk: "Orbitron" / fallback "Rajdhani"
    TikTok: "Anton" / fallback "Bebas Neue"
    Tarantino: "Courier New"
    Lo-fi: "VT323" / "Courier New"
    Vaporwave: "Press Start 2P" / "Orbitron"
  Otherwise use "context_aware" and the renderer picks at draw time.

RULE 5 — audioReactivity:
  enabled: true ONLY if user prompt mentions music/beat/sync/song
  or if the matching reference style is music-driven.
  Otherwise enabled: false.

RULE 6 — Every field in the schema MUST be present. Use sensible defaults:
  grade.offset: [0,0,0], grade.vibrance: 0, grade.hueShift: 0, grade.mix: 1.0
  grade.exposure: 0, grade.highlights: 0, grade.shadows: 0, grade.whites: 0, grade.blacks: 0
  grade.tealOrange: false, grade.bleachBypass: false, grade.splitToning: null
  timing.defaultSpeedMultiplier: 1.0
  All nullable fields (grain, vignette, bloom, chromaticAberration, etc): null if not applicable
